## Vanliga frågor

**Kan inte ansluta till rummet?**
Se till att alla enheter är på samma WiFi-nätverk eller har Bluetooth aktiverat.

**Hur många kan spela?**
Minst 3 spelare, max 8 spelare.

**Behövs internet?**
Nej, appen använder Bluetooth/lokalt nätverk. Ingen internetuppkoppling krävs under spelet.

**Appen kraschar eller beter sig konstigt?**
Stäng appen helt och starta om den. Kontakta oss gärna på mallraaisolutions@gmail.com om problemet kvarstår.
